# ShifaMind Dashboard

Minimalistic glassmorphism dashboard for **ShifaMind** — Concept-Grounded Clinical Diagnosis with Enforced Interpretability.

## Deploy to Netlify

1. Push this repo to GitHub
2. Connect to Netlify → select repo
3. Build settings: **Publish directory** = `.` (root)
4. Deploy

Or drag-and-drop this folder directly into [Netlify Drop](https://app.netlify.com/drop).

## Stack

- Pure HTML / CSS / JS — zero dependencies
- Google Fonts (Outfit + JetBrains Mono)
- Netlify config included (`netlify.toml` + `_redirects`)

## Structure

```
├── index.html        # Single-page dashboard
├── favicon.svg       # SVG favicon
├── netlify.toml      # Netlify build config
├── _redirects        # SPA fallback
├── robots.txt        # SEO
└── README.md
```
